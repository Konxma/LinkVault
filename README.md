# LinkVault

**LinkVault** — це десктопний агрегатор посилань, створений для зручного збереження, категоризації та швидкого пошуку вебзакладок. Проєкт розроблено в рамках навчальної практики з фокусом на багатошарову архітектуру, патерни проєктування та чистий код.

## 🚀 Основний функціонал
- **Безпечна авторизація та реєстрація**: Паролі користувачів безпечно хешуються алгоритмом SHA-256.
- **Організація за категоріями**: Створення необмеженої кількості папок для структурування посилань.
- **Система тегів**: Кожному посиланню можна призначити кілька тегів (зв'язок багато-до-багатьох) для зручної навігації.
- **Реактивний пошук**: Миттєва фільтрація посилань за назвою, URL або тегами завдяки "біндінгу" даних.
- **Асинхронність**: Всі операції з базою даних виконуються у фонових потоках, що забезпечує плавність роботи графічного інтерфейсу без блокувань.

## 🛠 Технічний стек
- **Мова програмування**: Java 21+
- **Графічний інтерфейс**: JavaFX (FXML + CSS стилізація)
- **База даних**: PostgreSQL (чистий JDBC, кастомний пул з'єднань на базі `ArrayBlockingQueue`)
- **Міграції БД**: Flyway
- **Збірка та управління залежностями**: Maven
- **Тестування**: JUnit 5, Mockito, H2 (In-Memory Database для інтеграційних тестів)

## 🏗 Архітектура та патерни
Проєкт побудовано за **тришаровою архітектурою** (Repository -> Service -> UI) із застосуванням MVC.
Використані патерни:
- **Data Transfer Object (DTO)**: Для безпечної передачі даних між шаром бізнес-логіки та UI.
- **Dependency Injection (DI)**: Для впровадження залежностей (наприклад, передача `UserRepository` в `UserService`).
- **Singleton**: Для створення єдиного пулу з'єднань з базою даних.
