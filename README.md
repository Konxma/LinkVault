# LinkVault 🔗

**LinkVault** — це десктопний агрегатор посилань (закладок), розроблений мовою Java з використанням JavaFX. Програма дозволяє зручно організовувати веб-посилання по папках (категоріях), додавати до них теги, швидко шукати потрібну інформацію та експортувати дані для подальшого аналізу.

## 🚀 Основний функціонал
* **Керування користувачами:** Реєстрація та безпечна авторизація (хешування паролів SHA-256).
* **Категоризація:** Створення, редагування та видалення папок для логічного групування посилань.
* **Керування закладками (CRUD):** Додавання посилань із зазначенням URL, назви та тегів. Редагування та видалення.
* **Розумний пошук:** Реактивна фільтрація посилань за назвою, URL або тегами в реальному часі.
* **Експорт даних:** Збереження обраної категорії посилань у формат Excel (`.xlsx`) для звітності.

## 🛠 Технічний стек
* **Мова:** Java 21+
* **Графічний інтерфейс:** JavaFX (з використанням FXML)
* **Архітектура:** MVVM (Model-View-ViewModel), Тришарова архітектура (Repository -> Service -> UI).
* **База даних:** PostgreSQL
* **Міграції БД:** Flyway
* **Доступ до даних:** чистий JDBC з використанням власного Connection Pool (на базі `ArrayBlockingQueue`) та патернів `Unit of Work`, `Identity Map`.
* **Збірка проєкту:** Maven
* **Утиліти:** Apache POI (для Excel), Project Lombok (через налаштування IDE).
* **Тестування:** JUnit 5, Mockito, In-Memory БД H2 (для інтеграційних тестів).
